Runnning on iOS:
  Just run npx react-native run-ios
  If you wanna install a native library cd into ios folder and run pod install

Running on Android:
  Open Android studio, go to configure > AVD manager, and launch a device emulator
  Run npx react-native run-android
  To run on an Android phone, make sure developer mode is enabled. Then run the adb reverse command below
  and then react-native run-android.

Running on iOS phone:
  Open ios/athlosNative.xcworkspace with XCode. Then click Product > Run. Instructions are here:
  https://reactnative.dev/docs/running-on-device

Any new screens created should be wrapped in the gestureHandlerRootHOC Higher order component
import { gestureHandlerRootHOC } from 'react-native-gesture-handler';

Android doesnt recognize 'localhost' but does 10.0.2.2. Ios Doesnt recognize 10.0.2.2 though.
  run adb reverse tcp:8080 tcp:8080 to get android to connect to localhost port 8080. Change the port # if this changes
  run adb devices to get list of devices
  run adb -s <device_name> reverse tcp:backend_port tcp:backend_port

Sometimes things get stale in cache after installing updating modules. Run 
  rm -rf node_modules/
  rm -rf package.lock.json
  npm i
  react-native start --reset-cache
to reset the cache
